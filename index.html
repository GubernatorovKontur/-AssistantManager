Помощник менеджера

Добро пожаловать в Помощник менеджера!

Это прототип инструмента, который упрощает работу менеджера по продажам: от планирования задач до закрытия плана. Вот что уже готово и что будет дальше: Подготовил Губернаторов Данила Алексеевич

Планер

Простой список задач для ежедневной работы.

Уже есть: Добавление задач, отметка выполнения, удаление, сохранение.

Будет: Дедлайны, приоритеты, фильтр невыполненных.

Письма

Конструктор писем для общения с клиентами.

Уже есть: Шаблоны для Маркета, предпросмотр, импорт/копирование.

Будет: Шаблоны ОФД/УВН, редактирование, новые категории.

Ссылки

Удобное хранилище полезных ссылок.

Уже есть: Категории, добавление, поиск, предустановленные ссылки.

Будет: Фильтр по тегам, сортировка, экспорт.

Советник

Гайд по продажам продуктов Контура.

Уже есть: Тарифы Маркета, ОФД, КЭП с советами, заметки.

Будет: Скрипты, возражения, мотивация.

Взаимодействия с другими подразделениями

Инструкции для работы с коллегами.

Уже есть: Чек-листы, шаблоны (лиды, внедрение, спецкейсы), заметки.

Будет: Примеры инцидентов, новые спецкейсы.

Стратегия закрытия плана

Аналитика и мотивация для выполнения плана.

Уже есть: Показатели, пул оплат, цели, геймификация (очки).

Будет: Редактирование оплат, сложная геймификация, интеграция с CRM.
